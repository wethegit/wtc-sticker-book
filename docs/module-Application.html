add-sticker

Casts the drop sticker event and adds the specified sticker. NB: The position values should be normalised to the canvas PRIOR to being sent to this listener. This can be done using something like:

    offset = $(@canvas).offset()
    left = (window.pageXOffset || doc.scrollLeft) - (doc.clientLeft || 0);
    top = (window.pageYOffset || doc.scrollTop)  - (doc.clientTop || 0);

    adjustedGlobalPosition = {x: position.x - offset.left + left, y: position.y - offset.top + top}
Type:
  • object
Properties:
Name Type Description
value DisplayObject:Sticker

The sticker element to add.
position Object

the position of the dropped sticker on the canvas
